# Fork Graph — 実装済み機能一覧

> Claude Code のセッション Fork 系譜を git-graph 風に可視化する VS Code 拡張。
> 「あの良い状態のセッション」を主観メモ付きで再発見し、ワンクリックで開き直せる。
> （詳細設計: [ROADMAP.md](ROADMAP.md) / 各実装記録: [fork-graph-ext/docs/implementation/](../fork-graph-ext/docs/implementation/)）

## グラフ表示

- **Fork 系譜の可視化**: `~/.claude/projects/` の全セッション（.jsonl）を読み、ユーザープロンプトを
  ノードとする木／森を git-graph 風レーンで描画。レーン割当は「本流が親レーンを継ぐ」原則で自動。
- **セッション間 Fork の正規化（canonicalize）**: Fork で uuid/sessionId が振り直されても
  timestamp＋内容で複製を 1 ノードに畳む。Fork 地点で再スタンプされる末端 1 件も
  安定 ID（promptId / requestId）で二次結合。読み込み順に依存しない。
- **ノイズ除去**: ツール結果の差し戻し・システム挿入（isMeta）・`<ide_opened_file>` 等の
  自動添付をブロック単位で除外し、人の発話だけをノード化。
- **/compact マーカー**: コンテキスト圧縮を実行した位置のエッジに 2 重波線（≈）を交差表示。
  `compact_boundary` の `logicalParentUuid` で親鎖を橋渡しし、圧縮後も木が割れない。
- **折りたたみ**: 分岐ノードの ▼/▶ で部分木を畳める。プレビュー文字数もスライダで調整可。

## ノード操作（右パネル）

- **メモ＋ピン**: ノードごとに主観メモと★ピンを保存（`globalState`、安定キーで永続化。
  VS Code 再起動・ファイル再読込・Fork 複製をまたいで同じノードに付く）。
- **Open in Claude Code**: クリックで該当セッションを再開。経路は 2 つ:
  - 公式拡張（`claude-vscode.editor.open`）
  - CLI（エディタ領域の新規タブでターミナルを開き `claude --resume`）— **デフォルト ON のトグル**で選択可
- **CLI 固定ケースの自動判定**: 公式 UI で削除済みのセッション（`hiddenSessionIds` を検知して
  破線ピル＋「削除済み」バッジ表示）と、別フォルダのセッション（そのフォルダの実 cwd で
  ターミナル起動）は公式経路では開けないため CLI 固定。
- **「いつ」の表示**: 発話時刻を `YYYY-MM-DD HH:mm（◯日前）` 形式で表示。

## 検索・フィルタ

- **テキスト検索**: プロンプト＋メモをインクリメンタル検索。ヒット以外は減光（位置＝系譜の文脈ごと見つける）。
- **日時検索**: 自前カレンダー（チャットのある日に ● マーカー）＋時刻範囲（from–to）。テキスト検索と AND。
- **★ピンだけ表示**: ピン付きノード＋祖先パスのみ表示し、レーンを圧縮。

## 管理機能

- **フォルダ選択**: チャットが存在するプロジェクトフォルダを新しい順に一覧（ラベルは実 cwd）、切替可能。
- **ソフトデリート／復元**: ノード＋子孫を Fork Graph の表示からだけ隠す（データは一切触れない）。
  ☰メニューの復元リストから戻せる。
- **復元プレビュー**: 復元前に該当の木 1 本を**別タブ**で表示。接続位置を強調・自動選択し、
  中身も Open in Claude Code で確認してから復元できる。
- **自動リロード**: `fs.watch` でセッションファイルの追記を検知し、常に最新状態を表示
  （メモ入力中は適用を退避して入力消失を防ぐ）。
- **タブの復元**: Developer: Reload Window や VS Code 再起動後もタブが消えない
  （`WebviewPanelSerializer`）。

## UI / その他

- **言語選択**: 英語（デフォルト）・日本語・中国語・韓国語。カレンダーの月名・曜日や相対時刻も追従。
- **状態の安定性**: 選択・折りたたみ・メモ・ピン・ソフトデリートはすべて安定キー
  （canonical key の SHA-1）で保持され、リロードやフォルダ切替で壊れない。
